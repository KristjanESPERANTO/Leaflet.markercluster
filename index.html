Leaflet.markercluster

Leaflet.markercluster groups nearby markers into visual clusters for better performance and UX when displaying dense datasets. It supports smooth animations, spiderfying at max zoom, and works with standard Leaflet layers including L.GeoJSON.

MIT Licensed – View on GitHub or read the Leaflet documentation.
