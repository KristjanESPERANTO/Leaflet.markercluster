All Features Demo

This example demonstrates the main clustering features all enabled together: animated marker addition, coverage display on hover, and zoom-to-bounds on click.

let markers = new L.MarkerClusterGroup({
  animateAddingMarkers: true,
  showCoverageOnHover: true,
  zoomToBoundsOnClick: true
});

// Add markers with animation
markers.addLayer(newMarker);

Features:

  • Animated addition: Markers fly in when added
  • Coverage on hover: Shows cluster bounds polygon
  • Zoom on click: Click clusters to zoom to their bounds
  • Add/remove markers dynamically
